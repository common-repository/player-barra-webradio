== Description ==
Player barra webradio é um plugin para que você possa inserir a url do player de sua webradio no topo ou rodapé do seu site, sem precisar alterar qualquer arquivo.

== Installation ==
1. Baixe e instale o plugin e, em seguida, ative.
2. No painel de adminstração do Wordpress acesse CONFIGURAÇÕES->Player Barra Opções, informe a URL do seu player
